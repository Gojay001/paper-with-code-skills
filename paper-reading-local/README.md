# paper-reading-local

本地临时论文精读目录，**已被 `.gitignore` 忽略**（本 README 除外），不会进入 git 仓库。

## 用法

1. 用 `paper-logic-reading` 技能生成 `{slug}.html` 与 `assets/{slug}/` 到此目录
2. 浏览器打开：`file://.../paper-reading-local/{slug}.html`
3. 满意后再迁入 `paper-reading/` 并更新 `paper-with-code-list.md`
